### Why This Happened

**Single Tree Problems (Even Optimized):**
- One perspective on the data
- Sensitive to training data specifics
- Can still overfit in subtle ways
- High variance

**Random Forest Solutions:**
- 100 different perspectives (trees)
- Each tree sees slightly different data
- Averaging smooths out errors
- Much more stable
- Lower variance

**The Magic of Averaging:**
```
Tree 1: Predicts $200,000 (off by +$5,000)
Tree 2: Predicts $192,000 (off by -$3,000)
Tree 3: Predicts $197,000 (off by +$2,000)
...
Tree 100: Predicts $194,000 (off by -$1,000)

Average: $195,000 (off by only +$200!)
```

### Full Working Code

```python
import pandas as pd
from sklearn.metrics import mean_absolute_error
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeRegressor
from sklearn.ensemble import RandomForestRegressor

# Load data
iowa_file_path = '../input/home-data-for-ml-course/train.csv'
home_data = pd.read_csv(iowa_file_path)

# Prepare features and target
y = home_data.SalePrice
features = ['LotArea', 'YearBuilt', '1stFlrSF', '2ndFlrSF', 
            'FullBath', 'BedroomAbvGr', 'TotRmsAbvGrd']
X = home_data[features]

# Split data
train_X, val_X, train_y, val_y = train_test_split(X, y, random_state=1)

# Model 1: Unlimited Decision Tree
print("=" * 50)
print("UNLIMITED DECISION TREE")
print("=" * 50)
tree_model_1 = DecisionTreeRegressor(random_state=1)
tree_model_1.fit(train_X, train_y)
val_predictions_1 = tree_model_1.predict(val_X)
val_mae_1 = mean_absolute_error(val_predictions_1, val_y)
print(f"Validation MAE: ${val_mae_1:,.0f}")

# Model 2: Optimized Decision Tree
print("\n" + "=" * 50)
print("OPTIMIZED DECISION TREE (100 leaves)")
print("=" * 50)
tree_model_2 = DecisionTreeRegressor(max_leaf_nodes=100, random_state=1)
tree_model_2.fit(train_X, train_y)
val_predictions_2 = tree_model_2.predict(val_X)
val_mae_2 = mean_absolute_error(val_predictions_2, val_y)
print(f"Validation MAE: ${val_mae_2:,.0f}")

# Model 3: Random Forest
print("\n" + "=" * 50)
print("RANDOM FOREST")
print("=" * 50)
rf_model = RandomForestRegressor(random_state=1)
rf_model.fit(train_X, train_y)
rf_val_predictions = rf_model.predict(val_X)
rf_val_mae = mean_absolute_error(val_y, rf_val_predictions)
print(f"Validation MAE: ${rf_val_mae:,.0f}")

# Final Comparison
print("\n" + "=" * 50)
print("FINAL COMPARISON")
print("=" * 50)
print(f"Unlimited Tree:  ${val_mae_1:,.0f}")
print(f"Optimized Tree:  ${val_mae_2:,.0f} (improved ${val_mae_1 - val_mae_2:,.0f})")
print(f"Random Forest:   ${rf_val_mae:,.0f} (improved ${val_mae_2 - rf_val_mae:,.0f})")
print(f"\nTotal Improvement: ${val_mae_1 - rf_val_mae:,.0f} ({(val_mae_1 - rf_val_mae)/val_mae_1*100:.1f}%)")
```

**Expected Output:**
```

UNLIMITED DECISION TREE
==================================================
Validation MAE: $29,653

==================================================
OPTIMIZED DECISION TREE (100 leaves)
==================================================
Validation MAE: $27,283

==================================================
RANDOM FOREST
==================================================
Validation MAE: $21,857

==================================================
FINAL COMPARISON
==================================================
Unlimited Tree:  $29,653
Optimized Tree:  $27,283 (improved $2,370)
Random Forest:   $21,857 (improved $5,426)

Total Improvement: $7,796 (26.3%)
```
